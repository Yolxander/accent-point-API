📚 API Usage Examples

cURL Example:

curl -X 'POST' 'http://localhost:8000/api/v1/convert-voice?device=cpu&normalize=true' \ -H 'accept: application/json' \ -H 'Content-Type: multipart/form-data' \ -F 'input_audio=@input.wav;type=audio/wav' \ -F 'reference_audio=@reference.wav;type=audio/wav'

JavaScript Example:

const formData = new FormData(); formData.append('input_audio', inputFile); formData.append('reference_audio', referenceFile); const response = await fetch('/api/v1/convert-voice', { method: 'POST', body: formData }); const result = await response.json(); // Use result.play_url for direct audio playback

HTML Audio Player:

<audio controls> <source src="/api/v1/play-voice/{conversion_id}" type="audio/wav"> Your browser does not support the audio element. </audio>
